# Dashboard UI 設計規劃

## 品牌色彩系統

根據北歐餐桌 Logo 提取的企業色：

| 色彩 | 色碼 | 用途 |
|------|------|------|
| 北歐藍 | `#8BA4BC` | 主色、導航背景、品牌識別 |
| 深藍 | `#5A7A9A` | 導航 hover、標題 |
| 陽光黃 | `#F5C242` | 強調色、重要按鈕、提示 |
| 橙色 | `#E67E4A` | 場地模組、警告 |
| 綠色 | `#6BAB73` | 員工模組、成功狀態 |
| 粉色 | `#E8A5B8` | 通知模組 |
| 白色 | `#FFFFFF` | 內容區背景 |
| 淺灰 | `#F8FAFC` | 卡片背景 |

## Tailwind 自訂色彩

```js
// tailwind.config.ts
colors: {
  nordic: {
    50: '#F0F4F8',
    100: '#D9E2EC',
    200: '#BCCCDC',
    300: '#9FB3C8',
    400: '#8BA4BC',  // 主色
    500: '#5A7A9A',
    600: '#486A8C',
    700: '#334E68',
    800: '#243B53',
    900: '#102A43',
  },
  accent: {
    yellow: '#F5C242',
    orange: '#E67E4A',
    green: '#6BAB73',
    pink: '#E8A5B8',
  }
}
```

## 按鈕樣式

- **主要按鈕**：`bg-accent-yellow text-nordic-800 hover:bg-yellow-400`
- **次要按鈕**：`bg-nordic-500 text-white hover:bg-nordic-600`
- **輪廓按鈕**：`border-nordic-400 text-nordic-500 hover:bg-nordic-50`

## 狀態標籤

| 狀態 | 顏色 |
|------|------|

| 進行中 | `bg-nordic-100 text-nordic-700` |
| 已完成 | `bg-gray-100 text-gray-700` |
